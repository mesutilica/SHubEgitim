Css Position ile Konumlandırma

Html elementlerini ekranda konumlandırmak için kullanılır.

Position Özellikleri

  1. static: varsayılan değer, eleman yerinde konumlanır
  2. relative: göreceli
  3. fixed: istenilen noktaya sabitlenir
  4. absolute: mutlak
  5. sticky: yapışkan

Position-Relative

position:relative olan öğe 4 konumdan etkilenir. sol, sağ, alt, üst
